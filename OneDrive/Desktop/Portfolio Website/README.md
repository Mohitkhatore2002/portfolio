# Mohit Khatore | Full Stack Java Engineer Portfolio

A premium, high-performance portfolio showcasing architectural thinking and scalable system design.

## 🏗️ Technical Architecture
- **Language**: Java 17+
- **Frameworks**: Spring Boot, Spring MVC, Spring Data JPA
- **Database**: MySQL (3rd Normal Form Design)
- **Frontend**: Vanilla HTML5, CSS3 (BEM), GSAP (Animations)
- **Philosophy**: Clean Code, SOLID Principles, MVC Architecture

## 🚀 Key Features
- **FAANG-Level Case Studies**: Detailed problem-solution breakdowns.
- **Engineering Approach**: In-depth explanation of architectural decisions.
- **Premium Performance**: Framework-less frontend for sub-200ms rendering.
- **Mobile First**: Fully responsive across all devices (tested down to 320px).

## 🛠️ Local Development
1. Clone the repository.
2. Open `index.html` in any modern browser.
3. For backend integration details, see the project case studies within the site.
